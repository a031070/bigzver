# Магическая битва (bigzver)

Консольная пошаговая RPG — битва двух магов с заклинаниями атаки, защиты и лечения.

## Как играть

```bash
python main.py
```

На каждом ходу игрок выбирает тип заклинания и само заклинание. Враг пока ходит вхолостую (заглушка). Игра заканчивается, когда у одного из участников заканчивается HP.

## Заклинания

| Тип | Заклинание | Цена маны | Эффект |
|---|---|---|---|
| attack | Avada Kedavra | 25 | урон 25 |

| attack | Импетус | 6 | урон 3 |

| defense | Spirit | 5 | защита +5 |

## Структура проекта

```
sprites.py      — класс Unit (HP, защита, мана)
spells.py       — класс Spell и словарь всех заклинаний
settings.py     — константы (подписи ходов)
game_input.py   — ввод хода игрока
main.py         — игровой цикл
tests/          — pytest-тесты (60 штук)
```

## Как создавалось

Проект воссоздан по транскрипту живого стрима по программированию, который вёл Сергей («Big Zver»). На стриме он в реальном времени писал эту игру с нуля, объясняя зрителям каждый шаг: классы, методы, структуру файлов, логику боя. Игру он обычно использует на уроках программирования для детей, но на стриме немного её усложнил.

Транскрипт был получен через ASR (faster-whisper) из аудиозаписи стрима, затем по нему восстановлен исходный код.
